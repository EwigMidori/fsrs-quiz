# Security Policy

## Supported Versions

This project is in early development. Only the latest `main` branch state should be considered supported.

## Reporting

If you find a security issue:

- do not publish exploit details in a public issue first
- open a private report through the repository owner if available
- if private reporting is not configured yet, open a minimal public issue without sensitive details and request a secure contact path

## Notes

- This project stores study progress in the browser via IndexedDB.
- Do not commit credentials, tokens, or private study data to the repository.
